# 🔍 OSINT Toolkit

A structured collection of OSINT tools, notes, and automation
scripts used during my CPEH certification path and penetration
testing labs.

---

## 📁 Structure

| Folder | Description |
|--------|-------------|
| `network/` | Network enumeration tools and notes |
| `dns/` | DNS reconnaissance tools and notes |
| `web/` | Web reconnaissance tools and notes |
| `passive/` | Passive reconnaissance tools and notes |
| `scripts/` | Bash automation scripts for recon phases |

---

## 🔁 Recon Phases Map

| Phase | Tools |
|-------|-------|
| Passive Recon | whois, theHarvester, Shodan, Google Dorks |
| DNS Recon | nslookup, dnsmap, dnstwist, dnsrecon, sublist3r |
| Network Recon | nmap, enum4linux, kerbrute |
| Web Recon | dirb, urlcrazy |

- Platform: UTM on MacBook Air M4
